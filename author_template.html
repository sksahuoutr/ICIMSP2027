Formatting Guidelines

  • Page Limit: 6–8 pages (including references, figures, and tables)
  • Format: Springer LNCS (Lecture Notes in Computer Science)
  • Font: 10pt Times New Roman (Word) or default LaTeX font
  • Paper Size: A4 (210mm × 297mm)
  • Margins: Follow Springer LNCS template specifications
  • Figures & Tables: Properly captioned and cited in text
  • References: Use Springer LNCS citation style

Important Notes

  • Use the template as provided — do not alter style definitions
  • Embed all fonts when converting to PDF (no font subsetting)
  • Remove any document protection or editing restrictions
  • Ensure all images have no transparent pixels (flatten if needed)
  • Keep PDF file size under 10 MB
  • Submit only original, unpublished work
  • At least one author must register for the conference
  • Papers must be submitted via the online submission system only
